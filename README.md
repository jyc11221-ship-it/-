# 风速序列预测 - 机器学习课程项目

基于多高度传感器采集的风速及相关气象数据，构建机器学习模型以预测未来时刻的风速。

## 📋 项目概述

- **任务类型**：时间序列预测
- **数据来源**：Hugging Face (10m/50m/100m 三个高度数据集)
- **预测任务**：
  - 单步预测：预测下一时刻风速
  - 多步预测：基于8小时历史窗口，预测未来1小时和16小时风速
- **对比模型**：Linear Regression、LSTM、Transformer

## 📊 评估指标

- MSE (均方误差)
- RMSE (均方根误差)
- MAE (平均绝对误差)
- R² (决定系数)
